# Comp-Graphic

Throughout this project, I developed a structured approach to designing software by breaking a large problem into smaller, manageable tasks. Rather than trying to build the entire 3D scene at once, I focused on creating one object at a time and gradually adding textures, lighting, and camera controls. This project helped me strengthen my ability to plan object layouts, select appropriate geometric shapes, and organize code into reusable functions. I followed an iterative design process by first recreating the basic scene, then refining each milestone with additional functionality such as textures, lighting, and navigation. This approach can be applied to future software projects because it allows complex problems to be solved in smaller, easier-to-test stages.

My approach to developing programs also evolved throughout the project. I learned the importance of writing modular code by creating separate rendering functions for each object, making the program easier to maintain and expand. I also relied heavily on testing after each change to ensure that new features did not introduce errors into the scene. Iteration played a significant role because each milestone built upon the previous one, allowing me to improve the project gradually rather than starting over. As the project progressed, I became more comfortable debugging graphics issues, adjusting transformations, applying textures, and fine-tuning lighting to achieve the desired appearance.

Computer science provides me with practical problem-solving skills that support my long-term educational and professional goals. Through computational graphics and visualizations, I gained experience with the graphics pipeline, object transformations, texture mapping, lighting models, and camera systems. These concepts strengthen my understanding of mathematics, programming, and computer graphics, providing a strong foundation for future coursework involving graphics, simulation, or visualization. Professionally, these skills could be applied to fields such as game development, simulation software, virtual reality, engineering visualization, and other industries that rely on interactive 3D environments. Even outside graphics programming, the planning, debugging, and iterative development skills I practiced during this project will benefit me throughout my software development career.
